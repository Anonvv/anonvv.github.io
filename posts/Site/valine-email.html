Valine 邮件提醒基础配置

27 / 09 / 2022

VitePress + Valine

由于官网更新后，早些的文档部分已经不适用，此篇就当前最新官网进行补充。

1. 登录 LeanCloud #

国内和国外版本都可以。

2. 设置环境变量 #

  • SITE_NAME : 网站名称。
  • SITE_URL : 网站地址, 最后不要加 / 。
  • SMTP_USER : SMTP 服务用户名，一般为邮箱地址。
  • SMTP_PASS : SMTP 密码，一般为授权码，而不是邮箱的登陆密码，请自行查询对应邮件服务商的获取方式。
  • SMTP_SERVICE : 邮件服务提供商，支持 QQ、163、126、Gmail、"Yahoo"、...... ，全部支持请参考 : Nodemailer Supported - services。 --- 如这里没有你使用的邮件提供商，请查看自定义邮件服务器
  • SENDER_NAME : 寄件人名称。

3. 部署 Git 项目 #

云引擎 > 部署 > 部署项目

填入项目地址：

https://github.com/zhaojun1998/Valine-Admin.git

注意： 由于当前源仓库配置的 node 版本是 6.x，过低可能会导致 npm 报错；所以如果遇到报错可以先 fork 这个项目到自己仓库，将 package.json 中的 node 版本改成 12.x 或以上，就可以成功部署了。

4. 点击部署按钮，开始构建。 #
