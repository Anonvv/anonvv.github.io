VitePress - Add Live2d

19 / 05 / 2022

1. Environment #

  • Vue3

2. Git repository #

live2d-model 🔗

3. Theme config #

Add a prop under themeConfig

// ~/.vitepress/config.js
themeConfig: {
    // custom proName
    live2d: true
}

3. Create a component #

or by CDN directly.

// L2d.ts

// live2d
import { h } from 'vue'

const L2d = () => {
    return(
        h('script', { src: 'https://cdn.jsdelivr.net/gh/stevenjoezhang/live2d-widget/autoload.js' })
    )
}

export default L2d

5. Render #

    <Content />
    <!-- any where -->
    <L2d v-if="theme.live2d" />
</template>

// import L2d ocmponent
import L2d from './L2d.ts'
</script>
