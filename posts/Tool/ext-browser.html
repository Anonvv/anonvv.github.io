浏览器插件推荐

23 / 08 / 2022

1. 安装方式 #

  • 能访问插件市场的用户，可以直接在插件市场安装，方便同步至浏览器账号；
    • Chrome：https://chrome.google.com/webstore/category/extensions
    • Edge: https://microsoftedge.microsoft.com/addons/Microsoft-Edge-Extensions-Home
  • 不能访问插件市场的用户可以下载 crx 文件手动安装，但无法同步到浏览器账号。
    • 插件 crx 下载：Chrome插件网, Cnplugins

2. 常规类 #

2.1. IGuge Helper(iGG谷歌学术助手) #

免费访问谷歌学术、谷歌搜索、Gmail、插件市场等谷歌基本服务。

2.2. Tampermonkey(油猴) #

用户脚本管理，可以安装各类丰富的扩展脚本。

2.3. AdGuard #

去除网页广告。

2.4. Grammarly #

英文写作语法检错、纠错和评分。

2.5. Vimium #

全键盘操作浏览器。

2.6. SuperCopy #

解除文本复制禁止。

3. 开发类 #

3.1. FeHelper(前端助手) #

JSON 代码美化。
