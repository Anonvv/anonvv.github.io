High-Frequency Git CMD

18 / 01 / 2022

1. Global config #

# global username
git config --global user.name "xxx"

# global email
git config --global user.email "xxx@email.com"

# show configured info
git config --global --list

# clean configured info
git config --global --unset [user.name]
