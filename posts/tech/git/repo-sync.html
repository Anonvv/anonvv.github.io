Synchronize Git and Gitee repository

25 / 07 / 2022

1. Create a remote repository on Git and Gitee respectively #

Such as:
GitHub: git@github.com:wonderven/test.git
Gitee: git@gitee.com:wonderven/test.git

2. Create and init a local repository #

~/test/

$ git init
1

3. Connect one of the two remote repositories #

$ git remote add origin git@github.com:wonderven/test.git
1

4. Open the config file under .git #

This is now:

[core]
     repositoryformatversion = 0
     filemode = false
     bare = false
     logallrefupdates = true
     symlinks = false
     ignorecase = true
 [remote "origin"]
     url = git@github.com:wonderven/test.git # github repo
     fetch = +refs/heads/*:refs/remotes/origin/*
 [branch "main"]
     remote = origin
     merge = refs/heads/main

5. Add another remote repository url #

~/.git/config

[core]
    repositoryformatversion = 0
    filemode = false
    bare = false
    logallrefupdates = true
    symlinks = false
    ignorecase = true
[remote "origin"]
    url = git@github.com:wonderven/test.git # github repo
    url = git@gitee.com:wonderven/test.git # gitee repo
    fetch = +refs/heads/*:refs/remotes/origin/*
[branch "main"]
    remote = origin
    merge = refs/heads/main
