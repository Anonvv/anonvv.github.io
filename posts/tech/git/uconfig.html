Multi-User Configuration

24 / 01 / 2022

Add the file config to ~/.ssh. #

# ~/.ssh/config

# site alias
Host github.com

# domain name
HostName github.com

# verify method
PreferredAuthentications publickey

# username(email better)
User user@email.com

# key file location
IdentityFile ~/.ssh/id_rsa
