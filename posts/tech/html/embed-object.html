HTML Tag [object] VS [embed]

09 / 02 / 2022

The two tags are both able to render media object such as audio, video, pdf and flash.
Due to the compatibility depending on different browsers, we should mainly use <object>.

1. <object> #

Supported by all major browsers

<object width="400" height="400" data="data path">
    rendered when the tag is not being supported.
</object>
1
2
3

2. <embed> #

Supported by all major browsers(no longer recommended).

<embed width="550" height="400" src="data path" />
1

3. Combination usage #

<object width="400" height="400" data="data path">
    <!-- Typically in IE -->
    <embed width="550" height="400" src="data path" />
</object>
