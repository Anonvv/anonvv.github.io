浏览器语音合成【Demo】

07 / 10 / 2022

1. SpeechSynthesisUtterance #

WARNING

Experimental: 这是一个实验中的功能
由于该功能对应的标准文档可能被重新修订，所以在未来版本的浏览器中该功能的语法和行为可能随之改变。

SpeechSynthesisUtterance 是 HTML5 中新增的 API, 用于将指定文字合成为对应的语音，也包含一些配置项，指定如何去阅读(语言,音量,音调)等。

2. Prototype #

PropDes
lang语言
pitch音调（值越高，声音越尖锐）
rate语速（值越大语速越快）
text文本
voice声音
volume音量（0-1）

3. Method #

PropDes
speak()将对应的实例添加到语音队列中
cancel()删除队列中所有的语音.如果正在播放,则直接停止
pause()暂停语音
resume()恢复暂停的语音
getVoices()获取支持的语言数组. 注意:必须添加在voiceschanged事件中才能生效

4. Demo #

js

speak('你好啊');
speak('hello world');

/**
 * 
 * @param {*} sentence 要说的句子
 * @param {*} pitch 音调：0 - 2, 默认：1
 * @param {*} rate 语速：0.1 - 10, 默认：1
 * @param {*} volume 音量：0 - 1，默认：0.5
 * 
 */
function speak(sentence, pitch=1, rate=1, volume=1, lang='zh-CN') {
    const utterance = new SpeechSynthesisUtterance(sentence);
    utterance.pitch = pitch;
    utterance.rate = rate;
    utterance.volume = volume;
    utterance.lang = lang;
    window.speechSynthesis.speak(utterance);
}

document.getElementById('pause').onclick = function() {
    window.speechSynthesis.pause(); // 暂停
}
document.getElementById('resume').onclick = function() {
    window.speechSynthesis.resume(); // 恢复
}
document.getElementById('cancel').onclick = function() {
    window.speechSynthesis.cancel(); // 取消
}
