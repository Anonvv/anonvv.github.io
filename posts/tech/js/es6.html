ES6+ 新特性

25 / 09 / 2022

面试备用

1. 基础 #

1.1. let & const #

  • let: 声明变量 代替 var 使用，声明的变量无法在“暂时性死区”中访问；
  • const: 声明常量 一旦声明，常量的地址（指针）不可被修改。

1.2. 模板字符串 #

使字符串连接方便易读。

const name = 'Tom'
const temp = `Hi, ${person.name}`

1.3. 箭头函数 #

特性：

  1. 箭头函数没有自己的 this，而是取决于上下文中的 this；
  2. 不能用 new 关键字调用，因为没有自己的 prototype 属性；
const fn = () => {
    console.log(this) // window
}

    return n * 10

1.5. 展开运算符(接收函数剩余参数) #

展开对象或数组

// 数组
const list = [1, 2, 3]
const list2 = [4, 5, 6]
console.log([...list, ...list2]) // 1, 2, 3
// 对象
const obj = { name: 'Tom', age: 10 }
const obj2 = {
    ...obj,
    gender: 0
}
console.log(obj2) // { name: 'Tom', age: 10, gender: 0 }

1.6. 对象/数组解构 #

// 对象
const person = { name: 'Tom', age: 10, gender: 0 }
const { name:newName, age } = person // 属性重命名 propName:newPropName
console.log(newName, age) // Tom 10
// 数组
const list = [1, 2, 3]
const [, b, c] = list // 不需要的位置使用,占位
console.log(b, c); // 2 3

1.7. 二进制 & 八进制整数字面量 #

  • 十进制，无前缀
  • 二进制，前缀 0b 或 0B
  • 八进制，前缀 0o 或 0O
  • 十六进制，前缀 0x 或 0X
const binary = 0b10 // 二进制 2
const octol = 0o10 // 八进制 8

1.8. Class & super关键字 #

Class & super：

// super class
class Person {
    constructor(name) {
        this.name = name
    }
    study() {
        console.log(`${this.name} is studying!`);
    }
}
// sub class（extends from Person）
class Student extends Person {
    constructor(name) {
        super(name) // call constructor of Person
    }
}
const s = new Student('Tom') // All person must to study!
s.study() // Tom is studying!

在对象中使用 super：

// super object
const person = {
    study() {
        console.log(`All person must to study!`);
    }
}
// sub object
const student = {
    study() {
        super.study()
    }
}
// 将 student 的 prototype 指向 person 的 prototype
Object.setPrototypeOf(student, person)

student.study()

for...of 遍历

2. 字符串新特性 #

2.1. includes(substr: string, index: number) #

从下标 index 开始，查询字符串是否包含子串(substr)，区分大小写。
返回：boolen

2.2. startsWidth(substr: string, index: number) / endsWidth(substr: string, index: number) #

从下标(index) 开始，查询字符串是否以指定子串(substr)开头/结尾，区分大小写。
返回：boolean

2.3. repeat(n: number) #

将原字符串重复 n 次。
返回：string 新字符串

2.4. padStart(lenth: number, str: string) / padEnd(length: number, str: string) #

填充函数。如果字符串不够长度 length，将用 str 将字符串补充到头部/尾部。 返回：新字符串

2.5. at(index: number) #

查找指定位置的字符，支持负索引。
返回：string 指定位置的字符

3. 对象新特性 #

3.1. Object.assign(target, source1...) #

将对象的所有可枚举属性复制到目标对象中。属于浅拷贝。
返回值：Object 拷贝的对象

3.2. Object.is(value1, value2) #

比较两个值是否严格相等(===)。
返回值：boolean

3.3. Object.keys(target: Object) / Object.values(target: Object) #

返回对象的所有属性/值。
返回值：Array

3.4. Object.entries(target: Object) #

返回对象键值对。
返回值：[][]
