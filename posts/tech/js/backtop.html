JavaScript 一键回到顶部

02 / 03 / 2022

1、获取页面滚动高度；

2、设置滚动高度为 0。

1. 环境 #

Vue3 + TS

2. 思路 #

  1. 创建组件插入页面，为控件添加点击事件(click)，并隐藏；
  2. 页面加载完成后(onMounted)，为window绑定滚动事件(croll)；
  3. 实时监听滚动高度(scrollTop)，超过设定高度值显示控件；
  4. 点击控件，滚动高度重置为 0，并隐藏控件；
  5. 为了节省性能开销，离开页面前，需要移除滚动事件。

3. 实现 #

<template>
    <div class="back-top" :style="{ display }" @click="backTop">
        <p>Backtop</p>
    </div>
</template>

<script setup lang="ts">
import { ref, onMounted, onUnmounted, computed } from 'vue'

const scrollHeight = ref(0) // 滚动高度
const display = computed(() => { // 大于设定值显示控件
    return scrollHeight.value > 400 ? 'block' : 'none'
})
// 页面加载完成
onMounted(() => {
    // 绑定滚动事件
    window.addEventListener('scroll', scrollFn)
})
// 页面卸载
onUnmounted(() => {
    // 移除滚动事件
    window.removeEventListener('scroll', scrollFn)
})
const scrollFn = (e) => {
    // 获取当前滚动高度
    scrollHeight.value = document.documentElement.scrollTop
}
// 点击返回顶部
const backTop = () => {
    window.scrollTo(0, 0)
}

</script>
<style lang="sass" scoped>
.back-top
    position: fixed
    bottom: 20px
    right: 20px
    text-align: center
    cursor: pointer
    z-index: 1
</style>
