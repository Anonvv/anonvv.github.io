CSS 常用单位

25 / 09 / 2022

首先来说一下对各种单位的定义。

  1. 绝对单位：是一个固定的值，反应了一个真实的物理尺寸。
  2. 相对单位：相对于参照元素的单位，随参照元素变化。

1. 绝对单位 #

1.1. px #

1px 对应显示器屏幕上的一个显示像素点。

2. 相对单位 #

2.1. em #

相对于该元素的 font-size 属性决定，1em = 1[font-size]。

2.2. rem(font size of the root element) #

相对于根元素(html)的字体大小决定。

2.3. % #

  1. 一般子元素： height 或 width 相对于直接父元素的 height 或 width；
  2. 非 relative 定位的子元素：
    • 子元素的 left 或 right 相对于直接非 static 定位父元素的 width；
    • 子元素的 top 或 bottom 相对于直接非 static 定位父元素的 height；
  3. 子元素的 padding 或 margin：垂直和水平方向都相对于直接父元素的 width；
  4. 子元素的 border-radius, translate, background-size：相对于元素自身的 width。

2.4. vw / vh #

相对于当前视口(view port)的宽度 / 高度。

2.5. vmin / vmax #

相对于当前视口宽度(vw)和高度(vh)的最小值（常用于 APP 横屏适配）。
