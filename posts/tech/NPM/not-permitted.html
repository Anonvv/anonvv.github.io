NPM：operation not permitted

07 / 03 / 2022

1. Error #

npm install

2. Solution #

  1. Run cmd as an administrator.

  2. Add global path in the file npmrc
    Open the npmrc file in the Nodejs installation directory.
    Ex: D:\Program\Nodejs\node_modules\npm\.npmrc
    Add flowing:

    prefix=D:\Program\Nodejs\node_global
cache=D:\Program\Nodejs\node_cache
