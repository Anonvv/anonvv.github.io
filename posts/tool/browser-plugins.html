22 / 09 / 2022
0

安装方式 #

  • 能访问插件市场的用户，建议在各自插件市场安装，可以直接同步到账号；
  • 不能访问插件市场的用户可以下载 crx 文件手动安装，但这样无法同步到浏览器账号。
  • 插件 crx 下载：
    • Chrome插件网🔗
    • Cnplugins🔗

1. 常规类 #

1. IGuge Helper(iGG谷歌学术助手) #

免费访问谷歌学术、谷歌搜索、Gmail、插件市场等谷歌基本服务。

2. Tampermonkey(油猴) #

用户脚本管理，可以安装各类丰富的扩展脚本。

3. AdGuard #

去除网页广告。

4. Grammarly #

英文写作语法检错、纠错和评分。

5. Vimium #

全键盘操作浏览器。
