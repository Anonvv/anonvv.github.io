reactive 声明式对象重新赋值丢失响应式的解决办法

28 / 09 / 2022

1. 使用 ref 声明变量 #

<script setup lang="ts">
import { ref } from 'vue'
let obj = ref({ // 声明变量
    name: 'Tom'
})
// 模拟修改操作
setTimeout(() => {
    obj.value = { // 重新赋值
        name: 'Coco'
    }
}, 1000);
</script>
    
<template>
    <div class="index">
        {{ obj.name }}
    </div>
</template>

2. 将数据保存到对象属性中 #

<script setup lang="ts">
import { reactive, toRefs } from 'vue'
let obj = reactive({
    data: {
        name: 'Tom'
    }
})
const { data } = toRefs(obj) // 解构到模板中直接使用
// 模拟修改操作
setTimeout(() => {
    obj.data = {
        name: 'Coco'
    }
}, 1000);
</script>
    
<template>
    <div class="index">
        name: {{ data.name }}
    </div>
</template>
