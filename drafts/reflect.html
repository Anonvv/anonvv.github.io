JavaScript 内置对象：Reflect

08 / 10 / 2022

Reflect 是一个内置的对象，它提供拦截 JavaScript 操作的方法； 和 Math 对象一样，Reflect的所有属性和方法都是静态的，不能用 new 调用。

1. Reflect.apply(target, thisArgument, argumentsList) #

Function.prototype.apply() 功能类似，调用一个方法并显示的指定其 this。 对一个函数进行调用操作，同时可以传入一个数组作为调用参数。和 。

参数：

ParamTypeDefaultDes
targetFunction-要调用的函数
thisArgumentObjectundefined要绑定的 this 对象
argumentsListArray,Object[]类数组，传入 target 函数的参数

返回值：target 函数调用结果 如 target 对象不可调用，抛出 TypeError。

示例：

console.log(Reflect.apply(Math.floor, undefined, [1.75]));
// expected output: 1
console.log(Reflect.apply(String.fromCharCode, undefined, [104, 101, 108, 108, 111]));
// expected output: "hello"
console.log(Reflect.apply(RegExp.prototype.exec, /ab/, ['confabulation']).index);
// expected output: 4
console.log(Reflect.apply(''.charAt, 'ponies', [3]));
// expected output: "i"

2. Reflect.construct(target, argumentsList[, newTarget]) #

类似 new 操作符，相当于执行 new target(...args)

参数：

ParamTypeDefaultDes
targetFunction-被运行的目标构造函数
argumentsListArray,Object-类数组，构造函数的参数
newTarget-target构造函数，作为创建的新对象的 constructor 属性，如果没有该参数，则默认创建的新对象的 constructor 属性为 target

返回值：以构造函数和参数始化的对象实例。

示例：

var obj = new Foo(...args);
var obj = Reflect.construct(Foo, args);

3. Reflect.defineProperty(target, propertyKey, attributes) #

和 Object.defineProperty() 类似，区别是 Reflect.defineProperty() 会返回调用成功或失败的 Boolean 值。

参数：

ParamTypeDes
targetObject目标对象
propertyKeyString要定义或修改属性的名称
attributesObject被定义或修改属性的描述对象

返回值：Boolean, 属性是否被成功定义。

const obj = {}
const flag = Reflect.defineProperty(obj,'a',{
    value : 1, //可选，默认undefined ，可以是任何数据类型
    writable : false,//可选，默认false，该属性的value是否能被改写
    configurable : true,//可选，默认为false 该属性描述符是否可配置 
    enumerable : false//可选，默认false 该属性是否能被枚举
})

if (flag) {
  // 成功
} else {
  // 失败
}

4. Reflect.deleteProperty(target, propertyKey) #

用于删除目标对象某一个属性，类似 delete 操作符，相当于执行 delete target[name]。

参数：

ParamTypeDes
targetObject目标对象
propertyKeyString需要删除属性的名称

返回值：Boolean，表明该属性是否被成功删除。

示例：

var obj = { x: 1, y: 2 };
Reflect.deleteProperty(obj, "x"); // true
obj; // { y: 2 }

var arr = [1, 2, 3, 4, 5];
Reflect.deleteProperty(arr, "3"); // true
arr; // [1, 2, 3, , 5]

// 如果属性不存在，返回 true
Reflect.deleteProperty({}, "foo"); // true

// 如果属性不可配置，返回 false
Reflect.deleteProperty(Object.freeze({foo: 1}), "foo"); // false

5. Reflect.get(target, propertyKey[, receiver]) #

获取对象某个属性的值，类似于 target[name]。

参数：

ParamTypeDes
targetObject目标对象
propertyKeyString需要获取值的属性名称
receiverObject如果target对象中指定了getter，receiver则为getter调用时的this值。

返回值：获取属性的值。

// Object
var obj = { x: 1, y: 2 };
Reflect.get(obj, "x"); // 1

// Array
Reflect.get(["zero", "one"], 1); // "one"

// Proxy with a get handler
var x = {p: 1};
var obj = new Proxy(x, {
  get(t, k, r) { return k + "bar"; }
});
Reflect.get(obj, "foo"); // "foobar"

6. Reflect.getOwnPropertyDescriptor(target, propertyKey) #

类似于 Object.getOwnPropertyDescriptor()。如果对象中存在该属性，则返回对应的属性描述符，否则返回 undefined。

7. Reflect.getPrototypeOf(target) #

类似于 Object.getPrototypeOf()。

8. Reflect.has(target, propertyKey) #

判断一个对象是否存在某个属性，和 in 运算符 的功能完全相同。

9. Reflect.isExtensible(target) #

类似于 Object.isExtensible().

10. Reflect.ownKeys(target) #

返回一个包含所有自身属性（不包含继承属性）的数组。(类似于 Object.keys(), 但不会受enumerable 影响).

11. Reflect.preventExtensions(target) #

类似于 Object.preventExtensions()。返回一个Boolean。

12. Reflect.set(target, propertyKey, value[, receiver]) #

将值分配给属性的函数。返回一个Boolean，如果更新成功，则返回true。

12.1. Reflect.setPrototypeOf(target, prototype) #

设置对象原型的函数。返回一个 Boolean，如果更新成功，则返回 true。
